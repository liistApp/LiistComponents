## Properties

- **`name`**: name of the place
- **`comment`**: comment to a place given by creator.
- **`thumbnail`**: image url
- **`tag`**: emoticon passed as string
- **`itemId`**: itemId of that place to be able to quickly fetch place details from firestore.
